# Date Pickers
Vue构建的日期选择组件
## 项目描述 ##
一个简单的时间选择器，构建为vue组件，可直接在vue实例中使用
## 项目文件 ##

datepickers.js --- vue组件文件

style.css --- 样式文件，可以按照自己喜欢重写样式

datePickers.vue --- vue单文件组件
